# Automation Factory

A simple TypeScript program that classifies packages as **rejected**, **special**, or **standard** based on their dimensions and mass.

---

## Prerequisites

Before running the project, make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v16 or higher recommended)  
- [TypeScript](https://www.typescriptlang.org/) compiler (`tsc`)

Clone project:

```bash
git clone git@github.com:MarleeG/automation-factory.git
cd automation-factory
```

Install dependencies
```bash
npm install
```

You can now run run the program while in the repo itself by running the command below. Test the changes
```bash
npm run start
```

To test the different values update the sort() functions four arguments on line 44 on the app.ts file and re-run the command above. 

This project uses Jest for testing. Run the test suite with:
```bash
npm test
```
